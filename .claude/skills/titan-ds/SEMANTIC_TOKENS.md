# Semantic Tokens

Use these `var()` names in CSS and JSX styles. Never use raw hex/rgb or `--color-*` primitives.

## Card
- `--card-background`
- `--card-border`
- `--card-radius`
- `--card-shadow`

## Button
- `--button-primary-bg`
- `--button-secondary-bg`
- `--button-tertiary-bg`
- `--icon-button-radius`

## Surface
- `--surface-page`
- `--surface-navbar`
- `--surface-0`
- `--surface-1`

## Spacing
- `--spacing-4xs`
- `--spacing-3xs`
- `--spacing-2xs`
- `--spacing-xs`
- `--spacing-s`
- `--spacing-m`

## Radius
- `--radius-control`
- `--radius-card`
- `--radius-dialog`

## Typography
- `--font-audiense`
- `--text-size-body`
- `--text-size-small`
- `--text-title`

## Input
- `--input-background`
- `--input-border`
- `--input-border-focus`
- `--input-text`
- `--input-placeholder`

## Table
- `--table-background`
- `--table-row-hover`
- `--table-border-bottom`

## Dialog
- `--dialog-background`
- `--dialog-radius`
- `--dialog-shadow`

## Drawer
- `--drawer-background`
- `--drawer-radius`
- `--drawer-shadow`

## Tooltip
- `--tooltip-background`
- `--tooltip-color`
- `--tooltip-radius`

## Focus
- `--focus-ring`
- `--focus-ring-offset`

---

To see the resolved values for a specific theme, call `titan_getTheme({ theme: "THEME", include: "css" })` via MCP.
